Overview

char variables can only store a single character. Strings refer to groups of characters, such as words/names/sentences. C++ does not have a built-in string type, but it does have a string class that you can include through a header file. (We'll see more on classes later)

Example Code: 

#include <iostream>
#include <string> // This is how we include the string class!
using namespace std;

int main()
{
    string username; // This is how we declare a string,
                     // no different from a regular variable!
    cout << "What is your name? ";
    cin >> username; // We can get strings from the keyboard.
    cout << "Oh, your name is "
            << username // We can print strings just like other variables.
            << "? That's kind of funny!\n";
}
