Printing Chess Board

Minor Program 1

tl;dr Students will use structs and 2D arrays to print a resemblance/model of a chess board.

Students will create a program which will define and print structures that may be commonly found in a game of chess. (Pieces, board) These do not need to be extremely functional as we are not making a game. (just printing the board)

There must be 2 structures, a board which contains a 2D array and a chess piece, and a piece which contains a symbol representation and position. There must be two functions, a new piece function which accepts a symbol, row, and column and returns a chess piece, and a print board function which accepts a chess board and returns nothing.

The students should print a few example boards/pieces, but I should easily be able to make and print my own in just a few lines. (see main() in the starter code at the bottom)

Topics Required

To complete this assignment, students will need to understand C++ functions, 2D arrays. Students will also need a basic understanding of structs and how to use them for aggregate (heterogeneous) types.

Purpose

Students completing this assignment will have reviewed and demonstrated earlier concepts from CS1, ensuring they are ready and able to move on to more advanced topics. Students will also have learned and demonstrated basic usage of structs, which is foundational to objects and critically important to the rest of this course.
