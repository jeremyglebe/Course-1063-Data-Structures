Overview

A struct, like an array, is an aggregate type, meaning it's composed of multiple objects. The difference is that structs are heterogenous, as opposed to homogeneous. This just means that while arrays contain several objects of the same type, structs often contain objects of different types.

Every object within a struct has it's own unique identifier. (As opposed to arrays where they are identified by index number) These internal objects are essentially the same as any other variable, but accessing them requires the usage of the dot operator. Remember, the dot operator is used on the variable of the struct type, not on the struct type itself. 

MyStructType variableName;
MyStructType.internalObjectName = SOME_VALUE; // Bad
variableName.internalObjectName = SOME_VALUE; // Good

The creation of a new struct type consists of using the struct keyword and specifying the types and names of it's internal objects. (The internal objects are sometimes called data members.) See the example code for the struct syntax. (Don't forget semicolon at the end of structs!)

Example Code: 

#include <iostream>
#include <string>
using namespace std;

struct BookType {
    string title;
    string author;
    int id;
    int publishedYear;
};

int main() {
    BookType frankenstein;
    frankenstein.title = "Frankenstein; or, The Modern Prometheus";
    frankenstein.author = "Mary Shelley";
    frankenstein.publishedYear = 1818;
    frankenstein.id = 1122;
    cout << frankenstein.title << '\n';
    cout << frankenstein.author << ", " << frankenstein.publishedYear << '\n';
}
