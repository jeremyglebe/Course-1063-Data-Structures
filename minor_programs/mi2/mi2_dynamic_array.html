Dynamic Array

Minor Program 2

tl;dr Students will complete a class that represents a "dynamic" array, which can be resized via a method in the class.

Students will use the starter code provided and implement needed code to make it work as intended.

Students will implement the class's constructor, making sure to initialize all variables to reasonable values.

Students will implement the class's destructor, ensuring no memory leaks occur.

Students will implement a method called resize() which will "resize" the dynamic array by creating a new array of a specified size, copying values over to the new array, and deleting the original array.

Students will not modify main. Students will not remove existing class data members.

Further detailed instructions are found in the starter code.

Topics Required

To complete this assignment, students will need to understand C++ classes, pointers, and dynamic memory allocation.

Purpose

Students completing this assignment will have demonstrated their knowledge of pointers, classes, and dynamic memory allocation.
